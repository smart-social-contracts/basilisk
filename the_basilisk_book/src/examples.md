# Examples

Basilisk has many example projects showing nearly all Basilisk APIs. They can be found in the [examples directory](https://github.com/demergent-labs/basilisk/tree/main/examples) of the [Basilisk GitHub repository](https://github.com/demergent-labs/basilisk).

We'll highlight a few of them here:

-   [Update](https://github.com/demergent-labs/basilisk/tree/main/examples/update)
-   [Primitive Types](https://github.com/demergent-labs/basilisk/tree/main/examples/primitive_types)
-   [Stable Structures](https://github.com/demergent-labs/basilisk/tree/main/examples/stable_structures)
-   [Cycles](https://github.com/demergent-labs/basilisk/tree/main/examples/cycles)
-   [Cross Canister Calls](https://github.com/demergent-labs/basilisk/tree/main/examples/cross_canister_calls)
-   [Management Canister](https://github.com/demergent-labs/basilisk/tree/main/examples/management_canister)
-   [Outgoing HTTP Requests](https://github.com/demergent-labs/basilisk/tree/main/examples/outgoing_http_requests)
-   [Incoming HTTP Requests](https://github.com/demergent-labs/basilisk/tree/main/examples/motoko_examples/http_counter)
-   [Pre and Post Upgrade](https://github.com/demergent-labs/basilisk/tree/main/examples/pre_and_post_upgrade)
-   [Timers](https://github.com/demergent-labs/basilisk/tree/main/examples/timers)
